HANDLING LONG BOQs:
==================
1. Use Excel's "Freeze Panes" feature:
   - Select cell B10
   - Go to View > Freeze Panes
   - This keeps project info visible while scrolling through items

2. The system will automatically:
   - Read all items from row 10 until it finds blank rows or "GRAND TOTAL"
   - Calculate totals regardless of number of items
   - Handle projects with 10 items or 1000+ items

3. For very long BOQs:
   - Use section headers (e.g., "GENERAL REQUIREMENTS", "SITE WORKS", "FINISHES")
   - Place section headers in column B with no item number
   - System will skip these rows automatically


COST BREAKDOWN FORMAT:
======================
Material Cost (Column I): Direct material costs only
Labor Cost (Column J): Direct labor costs only
Amount (Column L): Total cost = (Material + Labor) × Quantity

AUTOMATIC CALCULATIONS:
======================
Cost per sqm = Total Cost ÷ Lot Size
- System automatically calculates this when you upload the BOQ
- Displayed in preview and used for cost learning
- Requires Lot Size field to be filled

Examples:
- Site Clearing: Material=120, Labor=80, Qty=100 → Amount=20,000
- Excavation: Material=500, Labor=300, Qty=50 → Amount=40,000
- Concrete Work: Material=4,000, Labor=1,200, Qty=20 → Amount=104,000
- If Total Cost = 509,000 and Lot Size = 200 sqm → Cost per sqm = 2,545

IMPORTANT NOTES:
===============
1. Keep project info in rows 1-5 (fixed position)
2. Start BOQ items at row 10 (fixed position)
3. Use formulas for calculations (Amount = (Material + Labor) × Quantity)
4. All amounts in PHP (Philippine Peso)
5. Use standard UOM: sq.m, cu.m, pcs, kg, lot, lm, etc.
6. System auto-detects project information from header
7. Section headers help organize work by trade
8. Subtotal rows help with cost tracking
